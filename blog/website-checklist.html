The ultimate website checklist

Posted: Sat, 01 Jan 2011

The list of things to look before launch a new website

Accesibility

  • Opengraph
  • Accessibility check
  • Check with different resolutions
  • Check with Internet explorer

Social

  • Opengraph meta tags
  • Social buttons
  • Social button integration with google analytics

Monitor

  • Server monitoring
  • Google analytics

SEO

  • Create list of keywords
  • Check meta tags
  • Page titles
  • Sitemap.xml
  • Submit to search engines
  • Robots.txt
  • Build backlinks

Details

  • Favicon
  • Apple icons

Technical

  • 404 page
  • 500 page
  • Load test with blitz.io
  • Check redirect www to http://
  • Set backups
  • Opensearch
  • Validate CSS
  • Stop verbose error reporting DEBUG = FALSE
  • Check indexes in the database
  • Ensure semantical content

Load time

  • Validate (x)HTML
  • Compress/minimize js/css
  • Serve gzip
  • Optimize images
  • Use CDN
